Тестовий додаток на позицію  .NET Trainee
  Додаток зроблений на Windows Forms. Форма містить в собі дві кнопки. Для сереалізації та десереалізації об'єктів.
  Кнопка "Serialize", дозволяє сереалізацувати об'єкт, для цього потрібно вибрати потрібну дерикторію, і відразу ж потрібно буде вказати шлях, куди зберегти файл.
  Кнопка "Deserialize", дозволяє десереалізацувати об'єкт, для цього потрібно вибрати потрібний файл і відразу ж потрібно буде вказати шлях, куди його десереалізацувати.
